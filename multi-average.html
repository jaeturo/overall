여러 종목 평단가 계산기

여러 종목의 평균 단가를 한 번에 계산할 수 있습니다.
각 종목의 매입가와 수량을 입력하여 전체 포트폴리오의 평균 단가를 확인해보세요.

종목 관리

계산 결과

구분값
사용 방법 안내
  • 종목 추가: "+ 종목 추가" 버튼을 클릭하여 새로운 종목을 추가합니다.
  • 종목명: 각 종목의 이름을 입력합니다 (예: 삼성전자, SK하이닉스).
  • 매입가: 해당 종목의 평균 매입 단가를 입력합니다.
  • 보유 수량: 현재 보유하고 있는 주식 수량을 입력합니다.
  • 종목 삭제: 각 종목 옆의 "삭제" 버튼으로 불필요한 종목을 제거할 수 있습니다.
  • 모든 값을 입력한 후 전체 평단가 계산하기 버튼을 누르면 결과가 표시됩니다.
※ 계산 결과는 참고용이며, 실제 투자 전 전문가 상담을 권장합니다.
